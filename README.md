# ToDo list on RoR

#### [Link to test site](https://f-todo-list.herokuapp.com/)

### Run this app in local
  - cd /your_work_path
  - git clone https://github.com/OlegMarko/todo-on-rails.git
  - cd todo-on-rails
  - bundle install
  - rake db:migrate
  - rails s
  - open in your brouser http://localhost:3000/
